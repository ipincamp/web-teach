Instalasi PHP

Dalam Bagian Ini Kita Akan Memulai Dengan Instalasi PHP. Apa Itu PHP ? PHP adalah Bahasa Pemrograman Yang Disisipkan Kedalam HTML. PHP digunakan Oleh Developer Untuk Membuat Website Dinamis, PHP juga Sering Digunakan Untuk membuat Content Managemenet Service (CMS). Contoh Aplikasi Yang Menggunakan PHP : Facebook, Youtube , Wikipedia, Dsb !

Awali Semua Tutorial Dengan : sudo apt update && sudo apt upgrade

Instalasi PHP (Versi 8.0)

                                    # Tambah repository agar selalu update
                                    root@ubuntu:~$ sudo LC_ALL=C.UTF-8 add-apt-repository ppa:ondrej/php -y

                                    # Cek dengan update paket
                                    root@ubuntu:~$ sudo apt update -y

                                    # Instal dan beberapa dependensasi untuk Nginx
                                    root@ubuntu:~$ sudo apt install php8.0 php8.0-{cli,gd,mysql,pdo,mbstring,tokenizer,bcmath,xml,fpm,curl,zip} -y

                                    # Yay Kamu berhasil Mendownload PHP 8.0 Cek Dengan Command Line Ini
                                    root@ubuntu:~$ php -v

Instalasi NGINX

Apa itu NGINX ? Nginx Adalah web server dengan performa yang handal dan mempunyai beberapa fitur canggih lain yang mudah dikonfigurasi. Alhasil, Nginx mampu membuat website Anda lebih powerful dan canggih.

                                    # Instal langsung aja gaskeun!
                                    root@ubuntu:~$ sudo apt install nginx -y
                                    
                                    # Buka Firewall agar website dapat diakses (saya pakai ufw)
                                    root@ubuntu:~$ sudo ufw app list
                                    
                                    # Jika Anda ingin menggunakan Domain & TLS/SSL maka pilih yang "Nginx Full". Namun jika hanya ingin menggunakan lalu lintas web normal maka pilih yang "Nginx HTTP"
                                    
                                    # Buka Firewall dengan Domain & TLS/SSL.
                                    root@ubuntu:~$ sudo ufw allow 'Nginx Full'
                                    # Buka Firewall lalu lintas normal
                                    root@ubuntu:~$ sudo ufw allow 'Nginx HTTP'
                                    
                                    * Pilih salah satu saja, setelah itu update ufw.
                                    
                                    # Pastikan port untuk ssh tidak tertutup (Default 22).
                                    root@ubuntu:~$ sudo ufw allow ssh
                                    # Opsional.
                                    root@ubuntu:~$ sudo ufw allow 22/tcp
                                    
                                    # Aktifkan ufw.
                                    root@ubuntu:~$ sudo ufw enable
                                    
                                    # Cara melihat list rule ufw di Ubuntu.
                                    root@ubuntu:~$ sudo ufw status numbered
                                    
                                    # Cek apakah Nginx sudah berjalan dengan benar.
                                    root@ubuntu:~$ sudo systemctl status nginx
                                    
                                    # Cek di browser.
                                    http://ip_server_anda

Instalasi Redis

Apa Sih Redis Itu ? Dikuip Dari redis.io Redis adalah teknologi penyimpanan stuktur data dalam memori yang berfungsi sebagai database, pengelolaan cache, message broker dan thread.

                                    # Tambahkan repository agar selalu update.
                                    root@ubuntu:~$ sudo add-apt-repository ppa:chris-lea/redis-server -y
                                    
                                    # Cek dengan update paket
                                    root@ubuntu:~$ sudo apt update -y
                                    
                                    # Instal Redis Server
                                    root@ubuntu:~$ sudo apt install redis-server -y
                                    
                                    * Konfigurasi Redis Server
                                    # Gunakan nano sebagai editor di Ubuntu.
                                    root@ubuntu:~$ sudo nano /etc/redis/redis.conf
                                    
                                    # Lalu cari kata "requirepass" dengan ‎Shortcut "CTRL + W" lalu ketik "requirepass" dan hit Enter.
                                    Hapus tanda "#" dan akan menjadi seperti ini
                                    requirepass (password_redis_anda)
                                    
                                    # Contoh
                                    requirepass cfromd312
                                    
                                    # Setelah itu, save dengan Shortcut "CTRL + X" lalu klik "Y" dan "Enter"
                                    
                                    * Validasi Redis Server
                                    root@ubuntu:~$ sudo redis-cli
                                    root@ubuntu:~$ CONFIG SET requirepass 'password_redis_anda'
                                    root@ubuntu:~$ AUTH password_redis_anda
                                    root@ubuntu:~$ exit
                                    
                                    # Cek apakah sudah bekerja dengan normal.
                                    root@ubuntu:~$ sudo systemctl status redis

Instalasi MariaDB / MySQL

Apa Sih MariaDB / MySQL Itu ? MariaDB adalah software dari MySQL yang dikembangkan pada tahun 2009. Setelah Oracle Corp mengambil MySQL, dan ingin membuat stand-in. MariaDB memiliki SQL dalam DNA-nya dan memungkinkan pemrosesan data gaya ACID dengan atomisitas terjamin, daya tahan, isolasi, dan konsistensi untuk transaksi. Fitur Utama MariaDB adalah include support API JSON, replikasi data paralel dan berbagai mesin penyimpanan.

                            # Langsung instal MariaDB
                            root@ubuntu:~$ sudo apt install mariadb-server -y
                            
                            # Konfigurasi MariaDB
                            root@ubuntu:~$ sudo mysql_secure_installation

                            Enter current password for root (enter for none): Tekan "Enter" untuk skip

                            Remove anonymous users? Tekan "Y"

                            Disallow root login remotely? Tekan "Y"

                            Remove test database and access to it? Tekan "Y"

                            Reload privilege tables now? Tekan "Y"
                            
                            # Cek apakah sudah bekerja dengan normal.
                            root@ubuntu:~$ sudo systemctl status mariadb
