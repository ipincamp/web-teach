Persyaratan

PHP
Fungsi PHP adalah membuat atau mengembangkan situs web statis atau situs web dinamis atau aplikasi Web.
Rekomendasi agar menggunakan PHP v8.x, minimal PHP v7.4 atau versi yang lebih baru diatasnya.
MySQL
Fungsi dari MySQL adalah untuk membuat dan mengelola database pada sisi server yang memuat berbagai informasi dengan menggunakan bahasa SQL.
Rekomendasi agar menggunakan MariaDB v10.x (minimal MySQL v5.7.x atau MySQL v8.x.
Redis
Singkatan dari Remote Dictionary Server, adalah penyimpanan data nilai utama di dalam memori yang super cepat dengan sumber terbuka untuk digunakan sebagai database, cache, broker pesan, dan antrean.
Web Server
Berfungsi menerima permintaan HTTP atau HTTPS dari klien atau kita kenal dengan web browser (Chrome, Firefox). Ia juga akan mengirimkan respon atas permintaan kepada client dalam bentuk halaman web yang umumnya HTML.
Rekomendasi agar menggunakan Nginx.
Others
    curl
    Merupakan singkatan dari “Client URL”. Curl command dibuat untuk mengecek konektivitas ke URL dan juga sebagai tool transfer data.

    tar
    Merupakan singkatan dari Tape archive dan digunakan untuk mengompres serangkaian file dan folder. Umumnya, file yang telah di-compress dengan menggunakan command tar akan tersimpan dalam bentuk file .tar. Kompresi selanjutnya yang menggunakan gzip akan menghasilkan file dalam format .tar.gz.

    unzip
    Utilitas yang membantu Anda membuat daftar, menguji dan mengekstrak arsip yang terkompresi dengan ekstensi .zip.

    git

    Untuk mengatur versi dari source code program Anda dengan memberikan tanda baris dan kode mana yang ditambah atau diganti.

    composer

    Dependency manager khusus PHP yang memiliki fungsionalitas seperti Gem (Ruby) atau Maven (Java). Anda bisa menginstall suatu library melalui composer dan composer akan secara otomatis menginstall library lain yang dibutuhkan, tanpa perlu mendownload satu persatu.
